Gravity

A framework for beautiful websites

"Gravity is a framework for building rock solid HTML5 websites. It uses SASS to control styles, so make sure you download it before trying this framework. You'll find lots of useful stuff here including an ECMA5 shiv which allows you to use new JavaScript features in older browsers."Owain Lewis
